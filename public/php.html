What is PHP?

Features

  • Extensive community support.

Variable and Tokens

PHP code consists of various tokens, including keywords, identifiers, constants, strings, and operators. These tokens are the building blocks of PHP scripts.

Variables in PHP are used to store data. They start with a dollar sign ($) followed by an identifier. PHP variables are case-sensitive.

PHP Scope

Control Statement

In programming, sometimes we need to execute different blocks of code depending on a specific condition. Control Statements help developers control the flow of program execution. They decide which code should run and which code should skip based on conditions.

If else Statement

The if statement executes a block of code only when the given condition is true. If the condition is false, the else block will execute.

if (expression) { #true }
OR
if (expression) { #true } else { #false }

Control Flow

  1. The program first evaluates the if condition.
  2. If the condition result is true, the if block code will execute.
  3. After executing the if block, the program continues with the next statement.
  4. If the condition result is false, the if block will be skipped.
  5. In an if else statement, the else block will execute when the condition is false.
  6. After completion, program execution continues with the remaining code.
<?php

    $isLogin = false;

    if ($isLogin) {
        echo "Welcome UserName";
    } else {
        echo "Welcome Guest";
    }

    $isLogin = true;

    if ($isLogin) {
        echo "add logout";
    }
?>
Welcome Guest
add logout

Nested If Statement

A nested if statement means writing one if statement inside another if statement.It is useful when we need to check multiple conditions step by step.

if (condition) { if (condition) { // code } } else { // code }
OR
if (condition) {} else if (condition) {} else {} 
<?php

    $age = 23;

    if ($age >= 18) {

        }

        echo "Not Eligible";
    }

?>
Eligible for driving and other activities

Switch Statement

The switch statement is used when we need to compare one value with multiple possible values. It provides a cleaner way to write multiple else if conditions.

swtich (experssion) { case value1 : //code case value2 : // code default : //code }

Control Flow

  1. The program first evaluates the switch expression.
  2. The value is compared with each case value one by one.
  3. If a matching case is found, that case block will execute.
  4. After executing the case block, the break statement stops the switch.
  5. If no case matches, the default block will execute.
  6. After completion, program execution continues with the next statement.
<?php

    $day = "Monday";

    switch ($day) {

        case "Monday": echo "Today is Monday";
            break;

        case "Tuesday": echo "Today is Tuesday";
            break;

        case "Wednesday": echo "Today is Wednesday";
            break;

        default: echo "Invalid Day";
    }
?>
Today is Monday

For Loop

The for loop is used when we know how many times we want to execute a block of code. It repeats the code until the given condition becomes false.

for (initialization; condition; increment) { //code }

Control Flow

  1. The program initializes the loop variable.
  2. The condition is checked before every iteration.
  3. If the condition is true, the loop block executes.
  4. After execution, the increment/decrement operation runs.
  5. The loop continues until the condition becomes false.
  6. After completion, program execution moves to the next statement.
<?php

    for ($i = 1; $i <= 5; $i++) {

        echo "Number : " . $i . "<br>";
    }

?>
Number : 1
Number : 2
Number : 3
Number : 4
Number : 5

While Loop

The while loop executes a block of code as long as the given condition remains true.

while (condition) { //code }

Control Flow

  1. The program checks the while condition.
  2. If the condition is true, the loop block executes.
  3. After execution, the condition is checked again.
  4. The loop continues until the condition becomes false.
  5. If the condition is false, the loop stops.
<?php

    $i = 1;

    while ($i <= 5) {

        echo "Number : " . $i . "<br>";
        $i++;
    }

?>

Number : 1
Number : 2
Number : 3
Number : 4
Number : 5

Do While Loop

The do while loop executes the code block first and checks the condition after execution. It always runs at least one time.

do { //code } while (condition);

Control Flow

  1. The program executes the loop block first.
  2. After execution, the condition is checked.
  3. If the condition is true, the loop runs again.
  4. If the condition is false, the loop stops.
  5. The code executes at least one time even if the condition is false.
<?php

    $i = 1;

    do {
        echo "Number : " . $i . "<br>";

        $i++;

    } while ($i <= 5);

?>

Function

A function is a reusable block of code that performs a specific task. Functions help to reduce code repetition and make programs easier to maintain.

function functionName() { // code }
OR
functionName();

Function Flow

  1. The program finds the function call.
  2. The control moves to the function definition.
  3. The code inside the function executes.
  4. If the function returns a value, it sends the value back.
  5. After execution, control returns to the next statement.
<?php

    function message() {

        echo "Welcome to PHP Function";
    }
    message();

?>
Welcome to PHP Function

Function Return Value

The return statement sends a value back from the function. After return executes, the function stops.

function functionName() { return value; }

Return Flow

  1. The function starts execution.
  2. The code inside the function runs.
  3. The return statement sends a value back.
  4. The function execution stops.
  5. The returned value can be stored or used.

Cookies

A cookie is a small piece of data stored in the user's browser. Cookies are used to remember user information between requests, such as login status, preferences, and other settings.

setcookie(name, value, expire, path);

Cookie Flow

  1. The server creates a cookie using the setcookie() function.
  2. The cookie is stored inside the user's browser.
  3. The browser sends the cookie back with every request to the same website.
  4. PHP reads the cookie using the $_COOKIE super global array.
  5. The server uses cookie data to remember user information.
  6. A cookie can be removed by setting an expired time.
<?php

    setcookie(
        "theme",
        "dark",
        time() + 3600,
        "/"
    );
    echo "Cookie Created";

?>
Cookie Created

Access Cookie

Cookies can be accessed using the $_COOKIE array.

$_COOKIE['cookie_name']; 
<?php

    if(isset($_COOKIE["theme"])) {

        echo "Theme is " . $_COOKIE["theme"];

    } else {

        echo "Cookie not found";
    }   

?>

setcookie(name, "", time() - 3600);

Session

A session is used to store user information on the server. Unlike cookies, session data is not stored in the user's browser. Sessions are commonly used for login systems, authentication, and temporary data storage.

session_start();

Session Flow

  1. The program starts a session using session_start().
  2. PHP creates a unique session ID for the user.
  3. Session data is stored on the server.
  4. The browser keeps the session ID to identify the user.
  5. PHP accesses session data using the $_SESSION array.
  6. The session can be destroyed when the user logs out.

Create Session

$_SESSION['name'] = value;

SERVER Variable

The $_SERVER is a PHP super global variable that contains information about the server, request, and execution environment. It provides details like current file path, request method, browser information, and server details.

$_SERVER
OR
$_SERVER['keyName'];

Server Variable Flow

  1. The user sends a request to the web server.
  2. The server processes the request.
  3. PHP creates the $_SERVER array with request information.
  4. The program accesses required server data using keys.
  5. PHP returns the response to the user.
  • $_SERVER["PHP_SELF"] Returns current executing file name
  • $_SERVER["REQUEST_METHOD"] Returns request method (GET / POST)
  • $_SERVER["SERVER_NAME"] Returns server host name
  • $_SERVER["REMOTE_ADDR"] Returns user's IP address
  • $_SERVER["HTTP_USER_AGENT"] Returns browser information
<?php

    echo "Current File : "
    . $_SERVER["PHP_SELF"];


    echo "<br>";


    echo "Request Method : "
    . $_SERVER["REQUEST_METHOD"];


    echo "<br>";


    echo "Server Name : "
    . $_SERVER["SERVER_NAME"];


?>
Current File : /index.php
Request Method : GET
Server Name : localhost
